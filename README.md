# CodeAlpha Tasks

This repository contains the projects completed as part of the CodeAlpha program.

## Projects

### 1. Word Detection Game
- Interactive and user-friendly terminal-based game.
- Designed with an attractive UI for an engaging user experience.
- Challenges users to detect and identify words efficiently.

### 2. Stock Portfolio Tracker
- Allows users to store and manage stock investments.
- Tracks stock portfolio data and helps monitor holdings.
- Simple and efficient portfolio management system.

### 3. File Organizer
- Automatically organizes files based on their extensions.
- For example, `.jpg` files are moved to a dedicated `jpg` folder.
- Helps keep directories clean and well-structured.

### 4. Chatbots
#### Simple Chatbot
- Rule-based chatbot using basic input/output operations.
- Responds to predefined user queries.

#### API-Based Chatbot
- Advanced chatbot powered by APIs.
- Capable of generating dynamic and intelligent responses.

## Technologies Used
- Python
- File Handling
- APIs
